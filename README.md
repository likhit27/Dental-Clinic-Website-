# 🦷 Mani Dental Clinic Website

> Professional dental clinic website for **Dr. Yogesh Trivedi, BDS** — Udaipur, Rajasthan, India.

---

## 📌 Overview

A fully responsive, SEO-friendly single-page website for Mani Dental Clinic. Built with pure HTML, CSS, and JavaScript — no frameworks required. Designed for fast loading even on 3G networks, and accessible on all devices and browsers.

---

## ✨ Features

- **Multi-section single page** with smooth scroll navigation
- **Online appointment booking** form with date/time picker
- **9 dental services** showcased with descriptions
- **Tabbed Patient Info** — FAQ, Payment Options, Education, Visit Prep
- **Doctor profile** with credentials and accolades
- **Patient testimonials** section
- **Clinic gallery** (photo-ready placeholders)
- **Google Maps** embed slot
- **Floating Call & Book buttons** always visible
- **Mobile hamburger menu** for small screens
- **Social media links** — Facebook, Instagram, Twitter, LinkedIn, Google, WhatsApp
- **SEO meta tags** and Schema.org structured data ready
- **Google Analytics** placeholder slot

## 🚀 Deployment Guide

### Option 1 — Shared Hosting (Recommended)

1. Download `index.html`
2. Log into your hosting **cPanel → File Manager**
3. Navigate to `public_html/`
4. Upload `index.html` and rename it to `index.html` (if not already)
5. Upload your `assets/` folder alongside it
6. Visit your domain — site is live ✅

### Option 2 — FTP Upload

1. Open **FileZilla** or any FTP client
2. Connect using your hosting FTP credentials
3. Navigate to `public_html/`
4. Drag and drop `index.html` and `assets/`

### Option 3 — Antigravity Deploy

1. Open project in [antigravity.dev](https://antigravity.dev)
2. Click **Deploy / Publish**
3. Get your live URL instantly
